When running the project, select "Flocking" in the Main Menu to go to the Flocking Lecture Exercise. 
Within that scene, you can move the target around with your mouse.

All C# scripts for this exercise can be found in "*.zip\Assets\LE - Flocking". 
Behavior algorithms are in "Flock.cs". Required behaviors can be found in the ApplyRules() function.
