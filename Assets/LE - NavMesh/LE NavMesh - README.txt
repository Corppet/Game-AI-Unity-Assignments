When running the project, select "NavMesh" in the Main Menu to go to the NavMesh Lecture Exercise. 
You may need to scroll down to find the option. Within that scene, you can move the target around by left-clicking on the level 
with your mouse.

All C# scripts for this exercise can be found in "*.zip\Assets\LE - NavMesh\Scripts". 
Behavior algorithms are in "NMPlayer.cs".
